# 🌦️ Weather Verse Dashboard

A modern React-based weather application that displays real-time weather information and a 5-day forecast using the OpenWeather API.

---

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Current temperature and "feels like" temperature
- 🌤️ Weather description with icons
- 💧 Humidity, wind speed, wind direction, pressure
- 📊 5-day temperature trend chart (Recharts)
- 🤖 Smart weather advice & alerts
- ⚡ Loading and error handling states
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- React.js (Functional Components + Hooks)
- Axios (API calls)
- Recharts (Data visualization)
- OpenWeather API
- CSS / Tailwind styling
